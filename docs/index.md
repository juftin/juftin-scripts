# juftin-scripts

Helpful Python scripts by [@juftin](https://github.com/juftin)

## Installation

Using [pipx](https://pypa.github.io/pipx/)

```shell
pipx install juftin-scripts
```

Using pip

```shell
pip install juftin-scripts
```
